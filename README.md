
# كارينجو (Careingo) - دليل المطورين الشامل 🐱🚀

مرحباً بك في مشروع **كارينجو**، المنصة التفاعلية المخصصة للنمو الشخصي وتطوير العادات اليومية، والمطورة بواسطة **Artiatech Studio**. تم تصميم هذا التطبيق ليكون رفيقاً يومياً للمستخدمين في مسارات (اللياقة، التغذية، السلوك، والدراسة) بأسلوب ممتع ومحفز يشبه الألعاب.

---

## 🛠 التقنيات المستخدمة (Tech Stack)

- **Framework**: Next.js 15 (App Router).
- **Library**: React 19.
- **Styling**: Tailwind CSS + Shadcn UI (Custom Themes).
- **Database**: Firebase Realtime Database (RTDB) - تم اختياره لسرعة الاستجابة في التحديثات اللحظية للعدادات.
- **Authentication**: Firebase Auth (Email/Password).
- **Icons**: Lucide React.
- **Components**: Radix UI (Accordion, Dialog, Tabs, etc.).
- **Feedback**: نظام أصوات مدمج للتفاعل (Mixkit SFX).

---

## 🏗 بنية المشروع (Architecture)

### 1. قاعدة البيانات (Realtime Database)
تتبع قاعدة البيانات نموذج **User-Ownership**:
- `/users/{userId}`: يحتوي على بيانات الملف الشخصي (id, name, avatar, points, streak, age, weight, height, registrationDate).
- `/users/{userId}/trackProgress/{trackName}`: يحتوي على التقدم في كل مسار (المرحلة الحالية، المراحل المكتملة، تاريخ آخر إكمال).
- `/users/{userId}/dailyPoints/{date}`: سجل النقاط اليومي المستخدم في التقويم ولوحة المتصدرين.

### 2. نظام المسارات (Tracks & Stages)
يتم تخزين بيانات المسارات في ملف ثابت (`src/lib/challenges.ts`) لضمان سرعة التحميل:
- كل مسار يتكون من 30 يوماً (مرحلة).
- **نظام القفل اليومي**: لا يمكن للمستخدم البدء في يوم جديد إلا بعد إكمال اليوم السابق، وبشرط مرور يوم كامل (أو انتظار منتصف الليل)، ويستثنى من ذلك اليوم الأول.
- **بونص التبكير**: يبدأ من الساعة 5 صباحاً بـ 75 نقطة ويتناقص تدريجياً.

### 3. نظام الصحة (BMI System)
يتم حساب مؤشر كتلة الجسم (BMI) ديناميكياً بناءً على بيانات الملف الشخصي:
- يتم عرضه في الصفحة الرئيسية كبطاقة أداء صحي.
- يظهر تصنيف القيمة الرقمية في لوحة المتصدرين كدليل على "نجاح الـ BMI".

## 🚀 ميزات برمجية هامة للمطورين

### 🗓 التقنيات في التقويم (RTL Calendar)
تم تعديل الـ CSS الخاص بـ `react-day-picker` لدعم الاتجاه من اليمين لليسار (RTL)، مع تمييز الأيام المكتملة باللون الأخضر.

### 🏆 لوحة المتصدرين (Optimized Leaderboard)
تستخدم لوحة المتصدرين استعلام `query(ref, orderByChild('points'), limitToLast(100))` لضمان الأداء العالي، مع عرض القيمة الرقمية للـ BMI لكل مستخدم.

### 🔊 نظام الأصوات (Sound Management)
يتم إدارة الأصوات عبر `src/lib/sounds.ts` باستخدام مثيلات `Audio` معالجة مسبقاً لضمان التشغيل الفوري دون تأخير.
