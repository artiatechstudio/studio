# **App Name**: Careingo

## Core Features:

- Secure User Accounts & Profile: Enables user registration, login, and comprehensive profile management (personal information, avatar selection, language/theme preferences, privacy policy, terms & conditions, contact developer, logout, change password, delete account, user registration rank). This feature relies on Realtime Database for user data and preferences.
- Daily Progression System: Presents four distinct 30-stage 'care' tracks (Fitness, Nutrition, Behavior, Study), inspired by Duolingo's progression style. Users can view their stage progress visually (locked, open, completed indicators), enforced to one stage completion per track per day. Users can revisit completed stages but not advance prematurely.
- Dynamic Daily Challenge Content Tool: Utilizes a generative AI tool to provide engaging and unique daily tasks and content for the Nutrition, Behavior, and Study tracks. This tool adapts themes and suggestions to keep the user motivated throughout their 30-day journey in each track.
- Fitness Track & User-Defined Plans: Manages the 30 stages for the Fitness track. This allows users to track their progress and document their self-designed daily fitness activities, including scheduled rest days, in alignment with their personal goals.
- Performance Tracking & Leaderboard: Displays the user's current overall progress, daily completion streak, and their global ranking on a leaderboard. It also features a system for earning and showcasing achievement badges (Sharat) based on milestone completions.
- Progress Calendar & Reminders: Provides an interactive calendar view to visualize daily completion status, track streaks, and identify missed days. Users can set personalized reminders and notifications to help maintain daily engagement with their tasks.
- Resource Hub & Legal Information: A dedicated section offering general tips, health information, and advice. This includes accessible links to the Privacy Policy, Terms & Conditions, and a 'Contact Developer' section for support.
- Advertising System: Integrates an advertising system within the application to generate revenue, ensuring a non-intrusive user experience.
- Motivational Notifications: Sends personalized, timely, and encouraging notifications to users, reminding them of their progress, upcoming tasks, and celebrating milestones to boost engagement and consistency.
- AI Helper Character: Introduces an interactive, cartoonish, graphic, and animated AI helper character, inspired by Duolingo, that provides encouragement, tips, and celebrates user progress. This character will use reasoning tools to provide contextually relevant support and guidance throughout the user's journey, adapting its expressions and movements to enhance engagement.

## Style Guidelines:

- Primary color: A comforting blue-purple (#3D4A99). This hue was chosen to evoke reliability and a nurturing feel, balancing tranquility with purpose to support the app's focus on personal care and development. It provides a stable and encouraging foundation for the user experience.
- Background color: A muted, desaturated light purple (#F2EFF4). Visually linked to the primary color, this soft, almost off-white background ensures maximum readability and creates a serene, non-distracting environment conducive to learning and progress.
- Accent color: A vibrant violet (#A833DB). Positioned analogously to the primary on the color wheel, this accent provides a dynamic contrast in both brightness and saturation. It is strategically used to highlight interactive elements, calls to action, and celebratory progress indicators, adding an energetic and motivating touch without being overpowering.
- Headline and Body font: 'PT Sans' (humanist sans-serif). This choice reflects the app's focus on personal growth with its modern yet warm character, ensuring excellent readability and a friendly user experience across all text, from daily challenges to profile information.
- Utilize a consistent set of clean, minimalist line icons. Icons for stage progression (e.g., padlock for locked, lightbulb for open, checkmark for complete) should be universally understandable. Other icons should represent themes of health, learning, and self-improvement with a simple, illustrative style.
- Employ a clear, intuitive 'dashboard' style layout for the main screen, presenting the four tracks as prominent, interactive cards. Within each track, adopt a 'map' or 'path' layout to visually represent the 30 stages, emphasizing progression and next steps. Prioritize consistent spacing and clear hierarchy to guide user interaction and minimize visual clutter.
- Implement subtle and engaging animations to provide positive feedback. These include smooth transitions between screens, dynamic progress bar updates, satisfying visual confirmations for task completion, and celebratory effects when milestones or tracks are finished. Animations should enhance usability without causing delays.
